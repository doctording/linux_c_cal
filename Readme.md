linux c 实现 linux下的cal简单命令

* $./cal 
  输出当前时间年月的日历

* $./cal 2016
  输出某一年的日历

* $./cal 10 2016
  输出指定年月的日历

测试环境
 * os: ubuntu 14.04 32bit
 * gcc version 4.8.4 (Ubuntu 4.8.4-2ubuntu1~14.04.3) 
---

知识点

* 关于年月的一些函数

* printf输出，printf输出是可以改变输出内容的颜色的

* linux下 #include <time.h> 可以获取当前时间

* makefile书写, main函数参数
